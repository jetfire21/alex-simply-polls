== Description ==

Simply Polls is a simple plugin to create a poll easily and quickly. You can use our polls on sidebars, posts and pages

Features:

Fully Responsive Polls
Easy install and use
Polls plugin works perfectly with all versions of WordPress
Ability to use Polls on pages and posts
Ability to use Polls on sidebars
Ability to use more then one survey form on widgets, create as many Polls as you need
Unlimited number of Polls
Polls with Single choice answers
Tested on popular WordPress themes
Ajax load polls
Viewing the poll results in the admin panel

If you found any bug in our plugin or have a question contact us at freerun-2012@yandex.ru

== Installation ==

1. Open `wp-content/plugins` Folder
2. Put: `Folder: alex-simply-polls`
3. Activate `Simply Polls` Plugin
4. Go to `WP-Admin -> Simply Polls`

= General Usage (Without Widget) =

To embed a specific poll in your post/page, use [alex_poll id="1"] where 1 is your poll id.

= General Usage (With Widget) =

1. Go to `WP-Admin -> Appearance -> Widgets`.
2. You can add the Text Widget by clicking on the 'Add' link besides it.
3. After adding, to embed a specific poll copy [alex_poll id="1"] in field "Content", where 1 is your poll id.
4. Click 'Save'.

== Upgrading ==

1. Deactivate `Simply Polls` Plugin
2. Open `wp-content/plugins` Folder
3. Put/Overwrite: `Folder: alex-simply-polls`
4. Activate `Simply Polls` Plugin
